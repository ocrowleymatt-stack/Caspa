# CASPA Commercial Positioning

**Baseline:** `7bd7932`

## 1. One sentence

CASPA is a private creative production studio that turns messy source material into guided, saved, export-ready finished work — not just AI text.

## 2. First ideal user

A serious author or creator (novelist, memoirist, playwright) with **existing fragments** — notes, drafts, research, odd artifacts — who wants to **finish** something professional without losing originals.

## 3. Killer workflow

Upload sources → choose product (wizard) → Guide Me → draft (Novel Write Pro) → critique (Gold) → Saved Writing → safe apply → DOCX export.

## 4. vs ChatGPT

ChatGPT forgets the project, has no Source Library, no Book Map, no safe apply, no export pack, no intimacy controls, no rescue wizard. CASPA **remembers and protects**.

## 5. vs Scrivener

Scrivener organizes; it does not draft, critique, route AI, or rescue messy material. CASPA adds **intelligent production layers** on top of structure.

## 6. vs Sudowrite / Novelcrafter

They focus on prose generation. CASPA covers **ingestion → plan → draft → critique → archive → export** with local-first privacy and finish-roadmap tooling.

## 7. First 5-minute wow

Upload three mixed files (note + fragment + receipt) → classify in Sources → open Guide Me → run Trash to Treasure or NWP on one chapter → see output in Saved Writing with link.

## 8. Private beta tester first steps

1. Create project  
2. Add 3 source assets  
3. Complete Production Wizard  
4. Follow Guide Me once  
5. One NWP draft  
6. Open Saved Writing  
7. Export DOCX  

## 9. Marketable now

- Source Library + chaos ingestion  
- Trash to Treasure  
- Book Map + missing chapters  
- Saved Writing + safe apply  
- DOCX/Markdown export  
- Local Ollama + multi-provider  
- Adult fiction settings  

## 10. Hide until mature

- Show Factory / Orchestra / Music Lab stack  
- Wonder / Quality / Taste elevation suite  
- PDF export (if flaky)  
- Forge legacy  
- Global Gold Pipeline page (vs project Improve)  

## 11. Pricing/value argument

**Time-to-finished-manuscript** and **data safety** for professional creators. Position as studio subscription, not token chat — value = memory + pipeline + export + trust.

## 12. Proof before outsiders

- James usability pass unaided  
- 9.5+ critical QA sustained  
- Fast smoke green on every deploy  
- 3 beta users complete export without support  
- No P1 journey gaps in material → export path
